Las alumnas del máster en Big Data Science de la Universidad de Navarra Paula Sanjuan Campos e Inés Hernández Pastor hemos creado este proyecto con el objetivo de plasmar en formato web, mediante la librería "streamlit", una interfaz de usuario de modo que se pueda interactuar y ver información de seis pares de criptomonedas.

El código del archivo "PROYECTO_PYTHON.py" se conecta a la API de Kraken, desde donde se descargan los datos necesarios para el análisis. Tras la obtención de los pares de criptos y realizar alguna transformación, se obtienen indicadores estocásticos que posteriormente se grafican. Todo ello se ha estructurado mediante funciones y clases, donde además se ha añadido el manejo de errores y excepciones.

En el archivo "MEMORIA.pdf" explicamos de manera detallada mediante capturas de pantalla todas las partes de nuestro código.
